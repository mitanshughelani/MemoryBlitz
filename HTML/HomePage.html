Number Sequence Game

Instructions

  • Press Start button to begin the game.
  • The game will generate a random number.
  • User has to guess the number correctly.
  • After the user guesses the sequence correctly, a random number is added to the sequence.
  • The level increases as the sequence is guessed correctly.
  • The sequence has to be guessed correctly in order to get to next level.
  • If the sequence is entered incorrectly, the game ends.
  • The total score would be shown simultaneously.
